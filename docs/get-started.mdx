This guide shows you how to use the Elastic OpenTelemetry Distribution for Java to instrument your Java application and send OpenTelemetry data to an Elastic Observability deployment.

## Install

1. Download the latest release [here](https://mvnrepository.com/artifact/co.elastic.otel/elastic-otel-javaagent/latest).

1. Run the distro by using the `-javaagent:` JVM argument with the path to the distro jar:

  ```bash
  java -javaagent:/path/to/agent.jar \
  -jar myapp.jar
  ```
